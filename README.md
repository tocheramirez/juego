# Piedra, Papel o Tijera en Python ✊

Juego de piedra, papel o tijera contra la máquina al mejor de 3 puntos.

## ¿Cómo funciona?
- El primero en llegar a 3 puntos gana
- La máquina elige aleatoriamente
- El juego valida que ingreses una opción válida

## Requisitos
- Python 3.x

## Cómo ejecutarlo
```bash
python piedra_papel_tijera.py
```

## Ejemplo de uso
```
Marcador es: Jugador 0 - Maquina 0
Elige piedra, papel o tijera: piedra
¡Ganaste!

Marcador es: Jugador 1 - Maquina 0
```

## Aprendizajes
Este proyecto fue hecho para practicar:
- Módulo `random`
- Bucles y condicionales
- Validación de entradas
- Manejo de puntajes
